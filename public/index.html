Welcome to GHUB NOW

A micro-service that redirects to the repository page of given package name. Supports monorepos.

Usage

https://ghub.now.sh/<package-name>
https://ghub.now.sh/<package-name@next>
https://ghub.now.sh/<package-name@canary>
https://ghub.now.sh/<package-name@1.2.3>

Example

Try some of these: ghub-now, hela, koa-better-body, @hela/core, @babel/core, @now/node, preact, gatsby, assemble.

Why?

Because GHUB.IO is slow, inconsistent and fells down too much.
